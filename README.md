# AWS_3tier_architecture_terraform

Deploying a 3-Tier Architecture in AWS Using Terraform using below AWS Services.



### Resources need to be created / installed :

* Custom VPC

* 2 Subnets (Public)

* 1 Subnet (Private)

* 2 EC2 Instances

* Security Group

* Elastic IP

* NAT Gateway

* Internet Gateway

* Route Table

* Application Load Balancer

* Apache Webserver

* Dynamo DB
